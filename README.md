# Research Paper Summarizer

Technology Stack:
- Python
- Streamlit
- LangGraph
- Google Gemini API
- FAISS
- LangChain

Features:
- PDF Upload
- Research Paper Summarization
- Question Answering using RAG
- Keyword Extraction
- Conversation Memory
- Response Evaluation
- Hallucination Reduction
- Performance Tuning

Run:

streamlit run app.py
